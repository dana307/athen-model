What it does

📊

FCFF DCF valuation

Forecasts the operating model to free cash flow, builds the WACC from CAPM, and discounts to an intrinsic per-share value.

⚖️

Comparable companies

EV/Sales, EV/EBITDA, P/E, PEG, ROE, ROCE and Debt/Equity vs peers, with a median-multiple implied value.

🎲

Monte Carlo

10,000 simulations over uncertain growth, margin and WACC give a full distribution of value and P(undervalued).

🔥

Sensitivity heatmaps

The classic WACC × terminal-growth grid, so you can see how fragile or robust the thesis really is.

🚦

Risk engine

Auto-detects leverage, margin erosion, negative FCF, receivables/inventory build-up and debt spikes — Low / Med / High.

📄

Automated reports

One command produces an institutional DOCX/PDF research note with charts, scenarios and a recommendation.

📈

Portfolio analytics & optimization

Allocation, concentration (HHI), and mean-variance / max-Sharpe / risk-parity / Black-Litterman optimization.

🌍

Macro stress testing

Oil, rate, inflation, INR and recession shocks propagated through the valuation, with sector-specific transmission.

🎓

Actuarial layer

Merton default probability, Vasicek stochastic discount rates, scenario-weighted value and Bayesian updating.

Try it from the command line

pip install -r requirements.txt

python main.py --ticker RELIANCE --valuation        # DCF intrinsic value
python main.py --ticker RELIANCE --montecarlo        # 10k-sim distribution
python main.py --ticker RELIANCE --report --peers TCS,INFY   # DOCX + PDF note
streamlit run dashboard/streamlit_app.py             # interactive dashboard
